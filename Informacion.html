Socios de Negocio
Descripción:

En el Formulario de Socios de Negocios se presenta el Registro almacenado de dos tipos de Socios: Proveedores, clientes; Este Nos permitirá realizar las acciones sobre nuestra base de datos como el de Inserta un nuevo socio, Actualizar Datos incluso eliminar un registro.

Articulos
Descripción:

Se presenta el formulario Articulos, en el cual estaran disponibles todos los registros ingresados y almacenados para un mejor tratamiento en la base de datos la cual permite insertar un nuevo articulo de igual forma actualizar cualquier cambio que se necesite hacer o si se desea eliminar un articulo por cualquier motivo que se presente.

Pedidos
Descripción:

La gestion pedidos inicia cuando el cliente realiza un pedido y termina cuando recibe su producto, el formulario pedidos permite coordinar todo el proceso de tramitación, desde el ingreso del ID, Id del socio, la fecha del pedido, el detalle, sub total, total ISV, total, la fecha de entrega y estado del pedido. El formulario permite ingresar, actualizar, eliminar.

Facturas
Descripcion

El formulario facturas se ejecuta cuando un socio realiza un pedido de articulos y procede a realizar el pago es ahi donde se obtiene una factura, validando todos los datos correspondiente ID, NUMERO_FACTURA, ID_SOCIO, FECHA_FACTURA,DETALLE,SUB_TOTAL, TOTAL_ISV,TOTAL, FECHA_VENCIMIENTO, ESTADO este formulario tambien nos permite instertar, actualizar y eliminar una factura.
